# Sistema de Helpdesk — Gerenciamento de Chamados Técnicos

Sistema de gerenciamento de chamados técnicos desenvolvido com Python, SQL e Power BI.

## Tecnologias utilizadas
- **Python** — backend e lógica de negócio
- **SQL (SQL Server)** — banco de dados relacional
- **Power BI** — dashboard analítico

## Funcionalidades
- Cadastro e controle de chamados técnicos
- Acompanhamento de status (aberto, em andamento, resolvido)
- Banco de dados relacional para armazenamento dos chamados
- Dashboard com métricas de atendimento e tempo de resolução
